# Visual Studio FTE



可以的，而且**你有不只一種方式可以取得 Azure 的使用權限**。依照「用途」不同，公司內實務上主要分成 **3 大類**，我幫你整理成「怎麼拿、適合什麼情境、要注意什麼」。

------

## ✅ 方式一：Visual Studio FTE 訂閱（最簡單、個人學習用）

**適合你目前 PoC / 自學 / 個人實驗**

- **所有 Microsoft FTE 都可申請**
- 內含 **每月 USD $150 的 Azure 使用額度**
- 屬於 **個人訂閱（非公司專案）**
- 主要用途：
  - 自學 Azure
  - PoC / Demo
  - 技術實驗（VM、Storage、AKS、小型服務）

🔹 官方內部文件明確說明：

> All Microsoft FTEs are eligible for $150/month of Azure spend.    [[Activating...bscription | SharePoint\]](https://microsoft.sharepoint.com/teams/AELBootCamp/SitePages/Activating-Your-Azure-Visual-Studio-FTE-Subscription.aspx?web=1)

🔹 啟用重點（很多人會卡這裡）：

- **一定要用「個人 MSA（非 @microsoft.com）」綁定**
- 先用公司帳號登入 VS Portal，再指定個人 email 啟用    [[How to act...E benefits | Word\]](https://microsoft.sharepoint.com/teams/bamxholaatlai-900fundamentalscertificationtraining/_layouts/15/Doc.aspx?sourcedoc={872515A8-F9DC-415A-B416-202E84EE0DFC}&file=How to activate your Azure subscription as part of your FTE benefits.docx&action=default&mobileredirect=true&DefaultItemOpen=1)

📌 **優點**

- 不用找 manager
- 不走審批流程
- 立刻可用

📌 **限制**

- ❌ 不適合正式產品 / 團隊共用
- ❌ 不能放 production workload

------

## ✅ 方式二：申請「公司內部 Azure Subscription」（最標準、專案用）

**適合：MDEP / Team PoC / 專案 / 測試環境**

這是公司正式流程，**Azure 花費會走組織 Cost Center**。

### 你可以申請的類型

- **Internal Subscription**（最常見）
- 用途：
  - Internal PoC
  - Demo
  - 非 production service
  - Team shared environment

🔹 官方流程文件（Internal Subscription）
 [[Creating i...scriptions | SharePoint\]](https://microsoft.sharepoint.com/teams/azureinternal/Pages/Creating internal subscriptions.aspx?web=1) [[Azure Subs...tion Setup | SharePoint\]](https://microsoft.sharepoint.com/teams/csastartguide/SitePages/Azure-Subscription-Setup.aspx?web=1)

### 申請時你需要填的重點

- Subscription Purpose（Internal / Hybrid / Demo）
- 是否 production
- **預估每月花費（預設 $500）**
- **Reporting Manager 需核准**

📌 **優點**

- ✅ 正式、合規
- ✅ 可團隊共用
- ✅ 適合 MDEP / Android / Infra PoC

📌 **注意**

- 一定會走 **manager approval**
- 需要合理說明用途與花費

------

## ✅ 方式三：特定組織（如 MCAPS）Managed Environment

這個比較**情境限定**，例如：

- Sales / Consulting
- Demo / Support / Learning

🔹 文件說明    [[MCAPS Mana...t Overview | SharePoint\]](https://microsoft.sharepoint.com/sites/ManagedEnvironment/SitePages/MCAPS-Managed-Environment-Overview.aspx?web=1)

📌 **通常不適用一般 Engineering team** （除非你被納入該 org）

## 🧭 我給你的實務建議（依你現在的情境）

### 🔹 如果你現在是：

- AI Wiki / Gollum 替代方案 PoC
- Docker / Kubernetes / Azure 測試
   👉 **先用 VS FTE Azure $150/month** ✅

### 🔹 如果你要：

- Team PoC
- 之後可能變成正式方案
- MDEP / Android infra 共用
   👉 **直接申請 Internal Azure Subscription** ✅
